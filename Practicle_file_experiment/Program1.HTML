COURSE CURRICULUM

Introduction

This course provides a foundational understanding of programming and web technologies. It is designed to build logical thinking, problem-solving ability, and basic coding skills.

Duration: 16 Weeks (Including Lectures and Practical Sessions)

Semester I Subjects

  1. Introduction to Programming
  2. Data Structures
  3. Web Technologies
  4. Computer Networks
  5. Mathematics for Computing

Total Number of Lectures: 48 (Approx. 12 per subject)

Programming Languages Covered

  • C
  • C++
  • Python
  • HTML

Course Description

Introduction to Programming
Focuses on algorithmic thinking and fundamentals of structured programming.
Web Technologies
Covers basic website creation using HTML and CSS for layout and styling.
Data Structures
Introduces arrays, linked lists, stacks, and queues for efficient data handling.

Learning Outcomes

  • Develop structured problem-solving and coding abilities.
  • Create and format webpages using HTML and CSS.
  • Understand data organization through fundamental data structures.
  • Gain familiarity with tools used in programming and web design.

Software Tools Used

  • Visual Studio Code
  • Notepad++
  • Google Chrome Developer Tools

Reference Books

  • Programming in ANSI C – E. Balagurusamy
  • Data Structures Using C – Reema Thareja
  • HTML and CSS: Design and Build Websites – Jon Duckett

End of Curriculum
